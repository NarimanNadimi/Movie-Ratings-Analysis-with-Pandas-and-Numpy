# Movie Ratings Analysis

این پروژه یک مثال مقدماتی برای تحلیل داده‌های فیلم‌ها با استفاده از **Pandas** و **NumPy** است.

## توضیحات پروژه
- داده‌ها شامل فیلم‌ها، ژانر، کاربران و امتیازها هستند.
- تحلیل‌ها شامل:
  - تعداد کل فیلم‌ها
  - تعداد کاربران
  - میانگین امتیاز هر فیلم
  - فیلم‌ها و کاربران برتر
- هدف این پروژه تمرین کار با **DataFrame** و **Series** و همچنین Markdown در **Jupyter Notebook** است.

## ساختار پروژه
```
movie_ratings_analysis/
├─ data/movies.csv
├─ notebooks/movie_analysis.ipynb
├─ README.md
└─ requirements.txt
```

## اجرای پروژه
1. نصب نیازمندی‌ها:
```
pip install -r requirements.txt
```
2. باز کردن نوت‌بوک:
```
jupyter notebook notebooks/movie_analysis.ipynb
```
3. اجرای سلول‌ها و مشاهده نتایج

## نکته
- دیتاست فعلی کوچک است و برای نتایج واقعی‌تر می‌توان رکوردهای بیشتری به CSV اضافه کرد.
- Markdown موجود در نوت‌بوک به شما کمک می‌کند تحلیل‌ها را مستند و قابل فهم ارائه دهید.
